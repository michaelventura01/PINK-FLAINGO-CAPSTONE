# 🦩 PINK FLAMINGO CAPSTONE

> Capstone project for Big Data Specialization - San Diego University

## 📋 Project Description

This repository contains the final capstone project for the Big Data specialization at the University of San Diego. The objective is to analyze data from a fictional bike rental company called **"Catch the Pink Flamingo"** and propose strategies to increase its revenue.

## 🎯 Objectives

- Explore and visualize transaction and customer chat data
- Segment customers using clustering techniques
- Predict customer behavior with decision trees
- Analyze communication networks using graph databases
- Propose personalized business recommendations

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| **Python / Jupyter Notebook** | Data analysis and modeling |
| **PySpark** | Large-scale clustering |
| **KNIME** | Decision tree modeling |
| **Splunk** | Data exploration and visualization |
| **Neo4j / Cypher** | Graph analytics on chat data |

## 📊 Workflow

The project follows a complete data science pipeline:

1. **Data Exploration** (Splunk) - Initial analysis of chat and transaction data
2. **Data Preparation & Modeling** (KNIME) - Cleaning, partitioning, and decision tree
3. **Clustering** (PySpark) - Customer segmentation
4. **Graph Analytics** (Neo4j) - Communication network analysis
5. **Recommendations** - Personalized marketing strategies

## 📈 Key Findings

- Customers can be grouped into distinct segments based on spending and usage patterns
- Decision trees can predict customer loyalty and churn risk
- Chat data reveals communication patterns that correlate with customer satisfaction
- Personalized marketing strategies are more effective than generic campaigns

## 💡 Recommendations

The final analysis recommends:
- Targeted discount campaigns for specific customer segments
- Improved communication through preferred channels
- Retention strategies for high-value customers
